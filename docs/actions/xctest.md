# xctest


Runs tests on the given simulator







xctest |
-----|----
Supported platforms | ios, mac
Author | @dtrenz



**1 Example**

```ruby
xctest(
  destination: "name=iPhone 7s,OS=10.0"
)
```





<hr />
To show the documentation in your terminal, run
```no-highlight
fastlane action xctest
```
