# recreate_schemes


Recreate not shared Xcode project schemes







recreate_schemes |
-----|----
Supported platforms | ios, mac
Author | @jerolimov



**1 Example**

```ruby
recreate_schemes(project: "./path/to/MyApp.xcodeproj")
```





**Parameters**

Key | Description
----|------------
  `project` | The Xcode project
