# backup_file


This action backs up your file to "[path].back"







backup_file |
-----|----
Supported platforms | ios, android, mac
Author | @gin0606



**1 Example**

```ruby
backup_file(path: "/path/to/file")
```





**Parameters**

Key | Description
----|------------
  `path` | Path to the file you want to backup
